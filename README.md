# lego-set-prices
We aimed to develop an analysis system that examines trends in LEGO set prices over time in order to identify potential factors that influence pricing.  The main question guiding this project is: Do LEGO sets retain their value, and if so, why?  We integrated historical retail data with real-time secondary market info in this project to analyze historical and current LEGO pricing treds and popular themes.  Data was taken from Brickset API and BrickLink API.
